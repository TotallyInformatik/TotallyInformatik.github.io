By the way, here are some instructions on how to use the programs:

Once you download the file, extract the contents of the zip file by right clicking on the zip file and on "extract all". Then you can view the contents and start the exe file. If you want the exe file to be located on your desktop, you can right click on the exe file and click on create shortcut. Then you can drag the shortcut that just popped up and use that.

Moodle-Calendar (from March-April 2020)

This Program is made in python with various modules such as tkinter, datetime, BeatifulSoup4 etc. It is a calendar where you can plan your day and see the tasks from Moodle, but unlike the calendar that the known Website Moodle which serves for online education already provides, this calendar only shows the tasks that are relevant (i.e. no tasks will be displayed that you have already submitted, no tasks that were due some time ago, etc.), that way, you can plan your day more easily.
